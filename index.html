Walkthrough & Methodology Presentation

02 / Problem Statement

Why This Project?

📚

Textbook Limitation

Traditional Ayurvedic anatomy education relies on 2D diagrams and static illustrations that fail to convey spatial relationships.

🧠

Passive Learning

Lecture-based methods do not engage students in active, experiential learning — resulting in poor retention of complex anatomical concepts.

🔍

Accessibility Gap

There is no self-paced, interactive tool that bridges ancient Ayurvedic textual descriptions with modern visual understanding.

💡

The Opportunity

VR technology can transform abstract concepts into tangible, immersive spatial experiences — making learning intuitive and memorable.

03 / Objectives

Project Objectives

  • Develop a VR application that immerses the user in an educational 3D environment
  • Present Kala Shareera content through pre-baked animated sequences rendered directly within the 3D level
  • Implement hands-free, gaze-based navigation for complete accessibility
  • Support multiple deployment targets — Mobile VR, Standalone Headsets (Meta Quest), and Desktop
  • Create a modular architecture extensible to additional Ayurvedic topics
04 / Methodology

Design & Methodology

👁️

Gaze-Only Interaction

Users interact by looking at objects. A radial progress indicator fills on sustained gaze, triggering actions after 2 seconds. No controllers needed.

🎬

Baked Animated Sequences

Each module contains an animated sequence pre-baked into the 3D level. Only content visible within the camera viewport is rendered in real-time.

🌐

Cross-Platform Rig

A universal player rig detects the host platform at runtime — gyroscope on mobile, native XR on headsets, mouse on desktop — no manual switching.

09 / Key Features

Key Features

🎯

Reticle System

Idle → Hover → Dwell states. A radial progress ring fills before triggering interaction. Custom glow shader for visibility.

🌌

Immersive Environment

Dark space-like void with subtle ambient lighting. Floating UI elements with sine-wave animations and dynamic glow on hover.

📱

Universal Player Rig

Auto-detects platform — mobile gyroscope, Meta Quest native XR, or desktop mouse input. Zero manual configuration.

🎓

Tutorial Overlay

Visual onboarding on every launch. Teaches gaze-based interaction controls. Dismissible via gaze on close button.

🔄

Smooth Transitions

Fade-in/fade-out scene transitions with loading screens. No jarring cuts between environments.

🔊

Audio System

Ambient background audio, UI interaction sounds (hover, select), and per-module audio management for immersion.
